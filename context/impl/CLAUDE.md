# Implementation Tracking

Records what was built, what's pending, what failed.

## Conventions

- Start: `impl-overview.md`
- Per-domain: `impl-{domain}.md`
- Dead ends: `dead-ends.md` — prevents retrying failures (critical)
- Compacted history: `archive/`
- Update after every session

## Contents

| File | Domain |
|------|--------|
| `impl-overview.md` | Cross-domain status |
| `impl-brand-cleanup.md` | Brand/identity |
| `impl-documentation.md` | Docs |
| `impl-rtk-integration.md` | RTK integration |
| `impl-startup-experience.md` | Startup/onboarding |
| `impl-visual-theme.md` | Visual theme |
| `loop-log.md` | Build loop log |
| `archive/` | Compacted history |
