- Claude Code plugin ecosystem: 9,000+ extensions, 101 official, find-skills has 661K+ installs
- Competitor landscape: Cursor (73.6% VS Code base), Windsurf (#1 LogRocket ranking), Aider (43K stars, 88% self-written), Continue (32K stars), Copilot (20M users, $2B ARR)

- Viral CLI tools: Oh My Zsh (170K stars, word-of-mouth), Warp ($67M funding, 500K users), RTK (single viral post)
- Virality pattern: quantified productivity claims are #1 mechanism
- TUI frameworks: Bubble Tea (41.4K stars, Go), Ink (37.5K stars, JS/React, used by Claude Code itself)
- SDD: Thoughtworks identified as key 2025 practice, 3 maturity levels, tools: Kiro/Spec-Kit/Tessl
- SDD insight: clear specs save tokens AND reduce hallucinations
- awesome-cursorrules (39K stars) shows curation repos go viral
- Marketplace adoption: discoverability tooling (find-skills) is most-installed category
