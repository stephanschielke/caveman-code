# Kits

Define WHAT needs implementing. Implementation-agnostic.

## Conventions

- Start: `cavekit-overview.md`
- Requirements: R-numbered (R1, R2, R3...)
- Every R has testable acceptance criteria
- Never prescribe HOW — that's plans/
- Cross-reference related domains
- Decompose into subdirs when domain spans independent concerns
- Generate/update: `/ck:sketch`
